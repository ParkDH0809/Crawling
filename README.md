# Crawling

Java Jsoup, 정규표현식, File I/O를 활용해보기 위해 만들었습니다.

기능:
네이버 뉴스 카테고리 URL을 입력하면 해당 카테고리의 뉴스 내용을 종합해 10회 이상 사용된 단어를 출력합니다.
